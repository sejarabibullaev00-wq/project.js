# 🚁 Drone Simulator

Интерактивный 3D-симулятор дрона, построенный на **Three.js** и **Node.js**. Управляйте дроном в реальном времени, исследуйте трёхмерное пространство и наслаждайтесь реалистичной физикой полёта.

## ✨ Особенности

- **Реалистичная физика** — имитация движения дрона с инерцией и плавными поворотами
- **Интуитивное управление** — управление с клавиатуры для взлёта, посадки и манёвров
- **3D-окружение** — трёхмерный мир с ландшафтом, деревьями и зданиями
- **Погодная система** — реалистичное освещение и тени
- **Интерфейс в реальном времени** — отображение высоты, скорости и статуса
- **Камера от третьего лица** — динамическое слежение за дроном
- **Звуковые эффекты** — имитация звука вращающихся пропеллеров (опционально)
- **Кросс-платформенность** — работает в любом современном браузере

## 🛠 Технологии

- **Node.js** — серверная среда выполнения
- **Three.js** — 3D библиотека для рендеринга
- **Express** — веб-фреймворк для статической раздачи файлов
- **Socket.io** — передача данных в реальном времени (для многопользовательского режима)
- **Cannon.js** — физический движок (опционально)
